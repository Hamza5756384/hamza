# NOVA Laptops — Interactive Arabic Product Website

موقع عربي متجاوب لعرض منتجات اللابتوب، مبني بـ HTML وCSS وJavaScript فقط، من دون مكتبات خارجية للحركة أو الواجهات.

## التشغيل

افتح `index.html` مباشرة، أو شغّل خادمًا محليًا:

```bash
python3 -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

## أبرز الخصائص

- تصميم RTL عربي كامل.
- تمرير قصصي مع مشهد ثابت وتبدّل تلقائي للمزايا.
- مجسمات لابتوب مصنوعة بالكامل بـ CSS.
- مؤثرات حركة محسنة باستخدام IntersectionObserver وrequestAnimationFrame.
- مختبر ألوان تفاعلي.
- فلترة المنتجات وجدول مقارنة متجاوب.
- دعم لوحة المفاتيح وقارئات الشاشة.
- دعم `prefers-reduced-motion`.
- لا يعتمد على مكتبات JavaScript خارجية.

> المواصفات والأسعار والعلامة NOVA تجريبية لأغراض تصميم النموذج فقط.
